Patent search service

This page provides three patent search functions.

The information shown below is a brief description of the functions.

Also, the information obtained through the search can be downloaded as an Excel file.


Simple search

Simple patent search can store data at high speed through static crawling. The detailed functions of each function will be explained through sub-items.


Advanced search

The detailed search function performs dynamic crawling using Selenium, which is relatively slow, but performs a more powerful function. The detailed functions of each function will be explained through sub-items.


Apple search

Search Patently Apple function uses dynamic crawling to collect article titles and link information from Patently Apple website. It returned in the form of a data frame.
Although the information collection speed is slower than static crawling, there is little limit to the collection target because continuous work is possible.
Since dynamic crawling is utilized, information on all pages for search keywords can be collected. The detailed functions of each function will be explained through sub-items.
